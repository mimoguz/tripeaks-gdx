Un jeu de solitaire Tri Peaks open source avec plusieurs agencements disponibles. Les parties sont pour le moment générées aléatoirement.

Changes: 

- Version 1.1 added new effects and option to use different card back designs.

- Version 1.1 also changes how the game state is persisted. This change should solve issues related to reloading the previous state, such as statistics getting wiped.
